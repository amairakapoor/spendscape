# 💸 Spendscape – AR Finance App

## 🔍 Overview
Spendscape is an innovative Augmented Reality (AR) based finance application that helps users visualize and manage their expenses in an interactive and immersive way.

## 💡 Problem
Traditional finance apps present data in static charts and lists, making it difficult for users to truly understand their spending habits and financial patterns.

## 🚀 Solution
Spendscape leverages AR technology to transform financial data into visual, interactive elements in real-world space—making expense tracking intuitive and engaging.

## ✨ Features
- 📊 AR-based expense visualization  
- 🧾 Expense tracking and categorization  
- 📈 Interactive financial insights  
- 🎯 Budget monitoring  
- 🧠 Smart spending analysis  

## ⚙️ Tech Stack
- WebAR
- HTML
- Node.js
